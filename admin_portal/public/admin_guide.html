Overview

Getting Started

The Admin Portal lets you manage all content on the Employee Hub intranet through a visual interface. No HTML knowledge is required — every change is point-and-click.

🔗

Manage Links

Add, edit, remove, and reorder directory links, Quick Access chips, and Company Resources.

👥

Manage Users

Create and manage admin/editor accounts and control which areas each user can edit.

🎨

Icons & Customization

Choose from 30+ built-in icons or paste your own custom SVG for any link.

📋

Testing Tracker

View and manage beta testing submissions and reported issues from the team.

Logging In

1
Navigate to the Admin Portal Go to http://www.aqua-aerobic.net:5025/
2
Enter your credentials Enter your username and password, then click Login.
3
You're in! The portal will show your username and role in the top bar. Admin accounts see all tabs; Editor accounts only see areas they're assigned to.
ℹ️ Contact IT if you need a login created or your password reset.
Core Feature

Manage Links

The Manage Links tab is your main workspace. All changes to the Employee Hub directory, Quick Access chips, and Company Resources happen here.

Selecting an Area

Use the Select Area dropdown at the top to pick which section of the intranet you want to edit. Each area corresponds to a section visible to employees.

💡 Use the Quick Search box inside the editor to instantly filter through all links without scrolling.

Adding a Link

1
Scroll to the category Find the Category or Sub-Category where the link belongs.
2
Click "Add Link" A new row appears with fields for the link label, URL, and icon.
3
Fill in the details Enter the display name, the full file path or URL, and choose an icon from the dropdown.
4
Click "Save & Publish to Intranet" Changes go live immediately on the Employee Hub.

Editing or Removing a Link

  • Click directly into the label or URL field to edit it inline.
  • Use the icon dropdown to change the icon — pick from the list or choose Custom SVG to paste your own.
  • Click the 🗑️ Remove button to delete a link. This cannot be undone after saving.
⚠️ Always click Save & Publish after making changes — edits are not saved automatically.
Structure

Categories & Sub-Categories

The directory is organized into Categories (top-level folders) and optional Sub-Categories (nested folders within a category).

Adding a Category

1
Click "Add Category" Found at the bottom of the editor area.
2
Enter a name Give the category a clear, descriptive name that employees will recognize.
3
Add links and save Click "Add Link" inside the new category, then Save & Publish when done.

Adding a Sub-Category

Inside any Category, click "Add Sub-Category" to create a nested folder. Sub-Categories appear as collapsible sections within a Category on the Employee Hub.

Navigation

Quick Search Chips

Quick Search chips appear as clickable shortcuts at the top of the Employee Hub. Each chip links to a resource or launches AquaSearch.

Two Types of Quick Search

  • Standard URL — Links directly to a web page, shared drive path, or document.
  • AquaSearch — Launches AquaSearch pre-filled with the chip's keyword, helping employees find documents instantly.

Adding a Chip

1
Click "Add Quick Search Chip" In the Quick Search section of the editor.
2
Enter a label and type Set the chip label and choose Standard URL or AquaSearch from the dropdown.
3
Enter the URL or keyword For AquaSearch chips, enter the search keyword. For Standard URL, enter the full link.
4
Save & Publish The chip appears immediately on the Employee Hub.
Customization

Icons & Customization

Every link and category can have an icon. Choose from the built-in library or use a custom SVG.

Built-In Icons

The icon dropdown includes 30+ pre-made icons covering common topics: Folder, Document, Search, People, Engineering, Calendar, Plane, Training, and more. Simply select from the list next to any link.

Custom SVG Icons

1
Select "Custom SVG" from the icon dropdown A text box appears for you to paste your SVG code.
2
Paste your SVG code Copy raw SVG markup (starting with <svg>) and paste it in. Free SVG icons are available at heroicons.com and lucide.dev.
3
Save & Publish The custom icon appears on the Employee Hub.
💡 If an icon looks wrong or you accidentally overwrite it, use the "Restore Original Custom SVG" button that appears next to the dropdown after selecting Custom SVG.
Organization

Reordering Items

Every item in the editor has Move Up ↑ and Move Down ↓ buttons. Use these to rearrange:

  • Links within a category or sub-category
  • Categories within an area
  • Sub-categories within a category
  • Quick Search chips
⚠️ Remember to click Save & Publish after reordering to apply changes.
Admin Only

Manage Users Admin

The Manage Users tab is visible only to users with the Admin role. Use it to create, edit, and delete portal accounts.

User Roles

Role Can Do Cannot Do
Admin Edit all areas, manage users, view all tabs, delete tracker entries —
Editor Edit their assigned areas only Manage Users, Settings, delete tracker entries

Adding a New User

1
Go to the "Manage Users" tab Visible only to Admin accounts.
2
Fill in the Add New User form Enter a username, password, select a role (Admin or Editor), and choose which areas the user can edit.
3
Click "Add User" The new account appears in the user table immediately.

Editing or Deleting a User

  • Click Edit next to any user to update their username, role, or allowed areas.
  • Click Delete to permanently remove an account.
⚠️ You cannot delete your own account while logged in.
Account

Change Password

Any logged-in user can change their own password from the Change Password tab.

1
Click "Change Password" in the top nav
2
Enter your current password This confirms your identity before allowing a change.
3
Enter and confirm your new password Then click "Update Password".
Beta Testing

Testing Tracker

The 📋 Testing Tracker link in the top nav opens the Beta Testing Tracker in a new tab. As an admin, you have additional capabilities:

  • 🗑️ Delete any test submission or reported issue (regular users cannot delete)
  • ✏️ Edit any entry without needing an access code
  • 🔐 Admin Note — add an internal note to any entry (only visible when logged in as admin)
  • 💬 Delete comments and replies in the General Discussion board
  • 🗂️ Remove attachments on any issue
ℹ️ You must be logged into the Admin Portal first for the tracker to recognize your admin session.
Reference

Tips & Best Practices

  • ✅ Always Save & Publish after making changes — the green button at the bottom of the editor.
  • 🔍 Use the Quick Search filter inside the editor when working with large areas — it narrows results instantly without page reload.
  • 🖼️ Keep icon choices consistent — try to use the same icon for the same type of resource across areas (e.g. always use the Folder icon for directories).
  • 🔗 Use full file paths for network drives (e.g. \\aqua47\shared\...) and full URLs for web links.
  • 🔒 Share Editor accounts (not Admin) with department leads who only need to update their own section.
  • 📋 Check the Testing Tracker regularly during the beta period to stay on top of reported issues.
